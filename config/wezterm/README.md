# Wezterm commands

- Leader: CTRL + SPACE
- Navigation between panes: LEADER + h/j/k/l
- Navigation between tabs: LEADER + W
- Show projects: LEADER + P
- Show workspaces: LEADER + F
- Rename tab: LEADER + ,
- Rename workspace: LEADER + .
- Toggle zoom: LEADER + Z
